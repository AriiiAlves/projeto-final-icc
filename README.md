Em Windows, utilize o CodeBlocs para rodar, eu acho.

Em Linux, para instalar as bibliotecas, dê

    sudo apt update

    sudo apt install -y liballegro5-dev

Se ainda faltar algo, utilize:

    sudo apt install liballegro5-dev liballegro-audio5-dev liballegro-acodec5-dev

Após isso, simplesmente compile e rode com

    ./all
